# InfraGuardian360

> Full 360° IT infrastructure monitoring — self-hosted, Docker-native, zero licence cost.

[![Licence](https://img.shields.io/badge/Licence-Apache%202.0-blue.svg)](LICENSE)

## Quick start
```bash
git clone https://github.com/Gsingh3001/infraguardian360.git
cd infraguardian360
cp config/.env.example .env
docker compose -f docker/docker-compose.core.yml up -d
```

## What it monitors

Switches · Routers · Access Points · Printers · IP Phones · Servers · VMs · Kubernetes · AWS · Azure · GCP
